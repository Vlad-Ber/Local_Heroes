# Grannhjälpen (FESS)

Projektarbete på kursen Datorsystem med projektarbete (1DT003) 
(1DT096) våren 2020, Uppsala universitet.

## Kom igång

Setup
```
$ git glone https://github.com/uu-dsp-os-ospp-2020/dsp-fess.git
$ cd dsp-fess
$ npm install 
```

Then run

```
// Terminal #1
$ npm start
// Terminal #2 
$ node server.js
```

Then access via post `http://localhost:3000/`
